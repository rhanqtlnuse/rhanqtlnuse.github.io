在 Windows 上安装 Spacemacs

发表于 2020-05-31 更新于 2020-06-01

0    前置条件

你的系统中应该有 Git。

1    安装 Emacs

从清华大学开源镜像站下载 Emacs 的 Windows 版本。截止到 2020 年 5 月，最新的版本是 26.3，因此进入 emacs-26 这个目录，找到前缀为 emacs-26.3 的文件。

你会发现有 8 个文件：

i686 和 x86_64 是架构，简单地说就是 i686 是 32 位的，x86_64 是 64 位的，根据自己电脑的情况自行选择。

有 .sig 后缀的文件是 PGP 签名（PGP signature），用于验证文件的完整性。可以使用 GNU 的 GnuPG 验证。

根据官方的说法，有 no-deps 字样的文件和没有 no-deps 字样的文件的区别在于，后者包含大量可选的 Emacs 依赖项，Emacs 的许多可选功能会使用这些依赖项。这些可选的功能包括：

  • 显示多种类型（PNG、JPEG、GIF、TIFF、SVG）的内联图像
  • SSL / TLS 安全的网络通信（HTTPS，IMAPS 等）
  • HTML 和 XML 解析（内置的 EWW 浏览器需要）
  • 内置的压缩文本的解压缩

如果你不需要这些依赖项，则可以下载有 no-deps 字样的版本。

下载你需要的文件，将其解压到某个位置，文中以 C:\Program Files\Emacs-26.3 为例。

how-to-use-italic-and-bold-fonts-properly

发表于 2020-05-01

1    英文

在西文中斜体主要在以下场合使用：

  • 表示强调，唤起注意
  • 书籍名称、文章标题、船舶名称等
  • 表示引用
  • 外语，如英文文章中夹杂的法语（例：A splendid coq au vin was served. ）

当字体没有斜体时可以考虑使用伪斜体代替。打字机、手写等无法使用斜体的时候遇到上述情况使用下划线或引号加以区别。 另外，当斜体中又遇到上述情况时，使用正体加以区别。

2    中文

https://www.uisdc.com/bold-underline-italic-fonts

3    结论

总结一下：

  • 强调的部分应该是很小一部分，最好是几个单词 / 词语，否则过多的强调等于什么都没强调
  • 斜体和粗体能够产生强调的效果是因为它们干扰了读者
  • 避免使用伪斜体和伪粗体
  • 避免使用下划线作强调（URL 例外，因为已经成了约定）
  • 除了使用斜体 / 粗体进行区分之外，还可以使用底色进行强调（正式文本中不要这样做），英文中还可以用全大写字母进行强调

跟代码规范一样，没有唯一的方式，关键是建立自己的一套规则然后贯彻。
