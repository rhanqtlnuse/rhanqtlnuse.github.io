LeetCode 1250. 检查「好数组」

发表于 2020-03-19
难度：困难

题面见 LeetCode

直观

看上去没什么头绪：范围太大了，枚举子集还好说，但是找到一组「任意整数」感觉就很困难，因此，我们不能真的去尝试找一组任意整数，而应该是寻找使得这样一组任意整数存在的条件。

观察

尝试几组数：

    [12,5,7,23] => true -- 5 * 3 + 7 * (-2) = 1
    [29,6,10]   => true -- 29 * (-1) + 6 * 5 = 1
    [3,6]       => false
    [2,6,12]    => false
    [5,6,12]    => true -- 5 * 5 + 6 * (-4) = 1
    [12,18,30]  => false

是否发现了一点眉目？为 true 的几个，出现在结果中的元素分别为：

    [5,7]
    [29,6,10]
    [5,6,12]

它们的最大公约数为 1，而其余的几组，无论如何也找不到这样的组合。以最后一组为例：

    [12] [18] [30] [12,18] [12,30] [18,30] [12,18,30]

最大公约数均不为 1。

其实可以简单的通过数学公式演示一下。如果一组数 $x_1, x_2, \dots, x_n$ 有最大公约数 g != 1，那么它们可以表示成

    [k1g, k2g, ..., kng]

将每一个乘一个任意整数 ci，它们的和为

    (c1k1 + c2k2 + ... + cnkn)g

因为

    c1k1 + c2k2 + ... + cnkn

为整数，因此整个和一定是 g 的整数倍，所以不可能为 1
