在 WSL 上安装 Ruby 和 Ruby on Rails

发表于 2020-03-23 更新于 2020-04-24 分类于 Ruby ， Ruby on Rails
如题

1    准备工作

  • 安装 WSL
  • 在 WSL 中安装 Git，并且配置 user.name 和 user.email，生成 SSH key，添加到 GitHub 账户
  • 为了加快 apt 的速度，最好改用国内的源，比如清华镜像源
  • 安装 node.js

2    安装 Ruby 2.7.0

使用 rbenv 管理 Ruby 时，在 RubyMine 中设置 Ruby SDK：

  1. File / Settings / Languages & Frameworks / Ruby SDK and Gems
  2. 点击 +，选择 New remote…
  3. 选择 WSL
  4. 在下面的路径框中填入 /home/\/.rbenv/versions/2.7.0/bin/ruby
  5. 点击 OK
