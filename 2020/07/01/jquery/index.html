1    什么是 jQuery？

jQuery 是一个快速、小型且功能丰富的 JavaScript 库。通过易于使用的 API（可在多种浏览器中使用），它使 HTML 文档的遍历和操作、事件处理、动画和 Ajax 等事情变得更加简单。结合了多功能性和可扩展性，jQuery 改变了数百万人编写 JavaScript 的方式 [1]。
