LeetCode 990：等式方程的可满足性

发表于 2020-07-15
如题

题目地址

这道题我想到了两个角度：

  • 其一，如果把有相等关系的变量分成一组，可以通过并查集来做；
  • 其二，如果从相等关系的传递性角度来看，可以通过图遍历完成。

版本一

基于这样一个想法：

  • 如果出现 a == b 的形式，那么所有与 a 有相等关系的变量不应该与 b 有不等关系，所有与 b 有相等关系的变量不应该与 a 有不等关系；
  • 如果出现 a != b 的形式，那么所有与 a 有相等关系的变量不应该与 b 有相等关系，所有与 b 有相等关系的变量不应该与 a 有相等关系（没有办法判定所有与 a / b 有不等关系的变量与 b / a 应该有什么关系。比如 ["a != b", "b != c", "a == c"] 是可满足的，而 ["a != b", "b != c", "a != c"] 也是可满足的），具体是什么数学原理还有待研究。

结果：通过

  • 时间：8 ~ 16 ms 之间
  • 内存：~11 MB

复杂度分析：

  • 注意这里的 BFS 的复杂度是常数级别的
  • 遍历整个 equations 向量：$O(n)$

整体时间复杂度：$O(n)$

但是，8 ms 的耗时也只能打败 84.36% 的 C++ 提交（2020/7/15），说明还有更优的解决方案，这时渐进复杂度就派不上用场了。
