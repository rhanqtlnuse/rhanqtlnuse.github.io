【译】AIOps 平台

发表于 2020-06-08 分类于 AIOps
by Andrew Lerner

原文地址：https://blogs.gartner.com/andrew-lerner/2017/08/09/aiops-platforms/

AIOps 是一项新兴技术，可以解决我最喜欢的问题 —— 改善 IT 运维。因此，我请 Gartner 分析师 Colin Fletcher 撰写了有关该主题的访客博客。

大约在三年前，许多企业的 IT 运维部门的领导者放弃了对能帮助他们更好、更快、成本更低地完成工作的人工智能（AI）的投资，从而将自己置于“皮匠的孩子”1的不稳定角色。

我们从许多 IT 运维负责人那里听到了他们为业务涉众构建复杂到令人难以置信的大数据和高级分析系统的故事，然而他们自己使用的却是原始的、反应性的（reactive）红/黄/绿指示灯和人工步骤来帮助运行保持这些系统的正常运行所需的基础设施。此外，我们所有人都对生活中来自在线零售商、搜索提供商、虚拟个人助理和娱乐服务的动态推荐非常熟悉，talk about a paradox！

现在，我不会说自那时以来一切都发生了变化，但也确实发生了不少变化，而且情况变得更好。从那时起，我们看到了许多令人振奋的发展，包括人们对将人工智能（AI）广泛的能力应用于企业 IT 运维管理挑战的接受度和兴趣的迅速转变。

因此，我们引入了 AIOps（最初称为 Algorithmic IT Operations，现在称为 AI for IT Operations）的概念，以描述对这些技术的兴趣和投资的增长。我们对 AIOps 的正式定义是：

AIOps 平台利用大数据、现代机器学习和其他高级分析技术，以主动、个性化和动态的洞察力直接或间接地增强 IT 运维（监视、自动化和服务台）的功能。AIOps 平台允许同时使用多个数据源、数据收集方法、（实时的和深度的）分析技术和表示技术。

现在，那些 AIOps 的先驱者已经让我们看到了越来越多的成功案例。此外，新的 AIOps 供应商正在出现，越来越多的 IT 运维管理工具供应商创造了新的 AIOps 产品或相关增强功能（请参阅网络性能监控和诊断的魔力象限和连续配置自动化工具的市场指南）。

为了帮助人们了解这种动态的技术前景，我们刚刚发布了《AIOps 平台市场指南》！我们在其中描述了 AIOps 平台市场的现状。
