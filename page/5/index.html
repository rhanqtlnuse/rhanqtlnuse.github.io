linux-manjaro-getting-started

发表于 2019-07-05 分类于 Linux ， Manjaro

系统初始化

配置国内镜像源

国外的镜像源速度很慢，可以选择使用中科大、浙大镜像源

    $ pacman-mirrors -i -c China -m rank

同步数据库

需要先同步数据库才能使用 pacman 安装软件

    $ pacman -Syyu

安装 yay

    $ pacman -S yay

注意：使用 yay 安装软件时不能用 sudo

系统配置

安装输入法

    $ yay -S fcitx
    $ yay -S fcitx-googlepinyin
    $ yay -S fcitx-configtool

也可以安装 fcitx-sogoupinyin 或 fcitx-im

注意在安装 fcitx-sogoupinyin 时，需要 qtwebkit 和 qtwekbit-bin 二选一，这时候最好选择 qtwebkit-bin，否则需要编译。
