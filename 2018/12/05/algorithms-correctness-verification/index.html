算法正确性证明

发表于 2018-12-05 更新于 2019-09-08 分类于 算法 (Algorithms)

有时候我们写出一个算法，暂时没有条件（比如数据的限制/算力的限制/手边没有电脑等）进行实际的测试，这时候可以对其进行证明。

证明的基本方法是数学归纳法。下面先讲解数学归纳法，然后分别给出证明递归和证明迭代的例子给出一些例子。

例 1 证明 Euclid 算法（辗转相除法）正确

证明：对 $b​$ 进行数学归纳

  1. $b = 0​$ 时，显然正确
  2. 假设 $b < k$ 时算法正确，则有 $\text{Euclid}(a, k-1) = \text{GCD}(a, k-1)$，那么当 $b = k$ 时，根据 Euclid 算法有 $\text{Euclid}(a, k) = \text{Euclid}(k, a\text{ mod }k)$，由于 $a\text{ mod }k \le k - 1$，故 $\text{Euclid}(a, k) = \text{Euclid}(k, a\text{ mod }k) = \text{GCD}(k, a\text{ mod }k)$，再根据最大公约数的性质，有 $\text{GCD}(a, k) = \text{GCD}(k, a\text{ mod }k)$，所以 $\text{Euclid}(a, k) = \text{GCD}(a, k)$
