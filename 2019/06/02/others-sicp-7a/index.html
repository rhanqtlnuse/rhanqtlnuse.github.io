Case dispatch

  • 数值，比如 3，求值结果就是本身
  • 变量，比如 x，求值结果为环境中对应的值
  • ‘foo，在 eval 时已经被转换为 (quote foo) 的形式，求值结果为 foo 本身
  • lambda 表达式，比如 (lambda (x) (+ x y))，转换为内部表示 (closure ((x) (+ x y)) <env>)
  • cond 表达式：(cond (p1 e1) (p2 e2) ...)
  • 普通表达式，如 (+ x 3)
