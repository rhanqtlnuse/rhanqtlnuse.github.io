友元概述

在 C++ 中，没有像 Java 中一样统一的 toString() 方法，可是有时我们想要实现类似的功能，想像输出基本类型那样输出一个对象，该怎么做呢？

友元是 C++ 中一种机制，允许友元直接访问和修改类的私有对象

友元

友元关系需要注意以下几点：

  • 友元关系是单向的
  • 友元不受类的访问权限的控制
  • 《代码大全》中也提到，要谨慎使用友元，因为友元机制本身是为了性能对类封装性的破坏，所以需要在安全性和性能两方面进行权衡
