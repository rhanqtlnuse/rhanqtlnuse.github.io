摘要

题目描述

Given a sequence of K integers ${ N_1, N_2, …, N_K }$. A continuous subsequence is defined to be ${ N_i, N_{i+1}, …, N_j }$ where 1 $\le$ i $\le$ j $\le$ K. The Maximum Subsequence is the continuous subsequence which has the largest sum of its elements. For example, given sequence { -2, 11, -4, 13, -5, -2 }, its maximum subsequence is { 11, -4, 13 } with the largest sum being 20.

Now you are supposed to find the largest sum, together with the first and the last numbers of the maximum subsequence.

Input Specification

Each input file contains one test case. Each case occupies two lines. The first line contains a positive integer K (≤10000). The second line contains K numbers, separated by a space.

Output Specification

For each test case, output in one line the largest sum, together with the first and the last numbers of the maximum subsequence. The numbers must be separated by one space, but there must be no extra space at the end of a line. In case that the maximum subsequence is not unique, output the one with the smallest indices i and j (as shown by the sample case). If all the K numbers are negative, then its maximum sum is defined to be 0, and you are supposed to output the first and the last numbers of the whole sequence.

分析

这道题是浙大《数据结构》MOOC 的一道例题，但是原题中只要求输出最大子序列和，而本题中需要输出相应的起始、结尾元素，增加了一些难度。

分情况进行如下的详细讨论：

  • 如果所有的元素都是负数，那么最大子序列和为 0
  • 如果有 1 个正数 x，那么最大子序列和为 x
  • 如果有 2 个正数 x 和 z，如果二者相邻，那么最大子序列和为 x+z，如果二者不相邻，那么最大子序列和取决于中间的其他负数，假设 x 和 z 中间的所有负数的和为 y，那么：
    • 如果 $x + y + z \ge z$，即 $x + y \ge 0$ 时，$x+y$ 对最大和有增益，应该保留
    • 如果 $x + y + z \lt z$，即 $x + y \lt 0$ 时，$x+y$ 会降低最大和，所以应该丢弃，即有最大和的子序列至少应该从 $z$ 开始
    • 需要注意的是，当 $x+y = 0$ 时，其实也可以将 $x + y$ 丢弃，但是题目要求输出起始位置 $i$ 和结束位置 $j$ 最小的元素，所以在这里需要保留
  • 有 2 个以上正数时，结果上面类似

如何检测是否全为负数？根据上面的讨论，如果存在正数，那么结果就应该是正数，所以当结果为负数时，说明全为负数，就可以按照全为负数的方式处理。

注意到上面的讨论没有说到 0 的情况，下面讨论一下：

  • 如果全为 0，那么输出应该是 0 0 0，可以忽略；
  • 如果数组的后半部分全为 0，那么：
    • 如果前半部分全为负数，那么输出为 0 0 0，可以忽略
    • 如果相当于在上面讨论的情况（1 个正数、2 个正数）中加入结尾的 0，结果不变：0 不会影响最大和，所以可以加到和中，但是结束元素应该保持不变
  • 如果数组的前半部分全为 0，那么：
    • 如果相当于在上面讨论的情况中加入起始的 0，那么起始元素应该更新
